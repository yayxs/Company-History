# YouTube

## 全球最大的横屏视频网站是？

全球最大的横屏视频网站毫无疑问是 YouTube。
它由Google拥有，是世界领先的视频分享平台，拥有超过20亿的月活跃用户。无论是从用户数量、视频上传量还是观看时长来看，YouTube都遥遥领先于其他所有以横屏内容为主的视频网站。
虽然现在有像TikTok这样的平台普及了竖屏视频，但YouTube仍然是全球用户观看和分享横屏视频内容的首选和最主要的目的地。

## YouTube哪一年成立，是全球最早的横屏视频网站吗？是谷歌创建的吗？
1. YouTube是哪一年成立的？

YouTube成立于 2005年2月。

历史上第一个上传到YouTube的视频是《我在动物园》（Me at the zoo），由其联合创始人贾德·卡林姆于2005年4月23日上传。

2. 它是全球最早的横屏视频网站吗？

它不是全球最早的横屏视频网站。

在YouTube之前，已经存在一些视频分享网站，其中最著名的是 Vimeo，它成立于2004年，比YouTube早一年。Vimeo从一开始就更侧重于为视频创作者和电影制作人提供高质量的托管服务。

但YouTube的关键成功之处在于，它极大地简化了视频上传、分享和嵌入到其他网页的过程，使其迅速普及并成为市场主导者。可以说，YouTube是第一个将大众化视频分享变为现实并取得巨大成
功的网站。

3. 是谷歌创建的吗？

不是，YouTube不是由谷歌创建的，而是被谷歌收购的。

这是一个非常常见的误解。YouTube的创始人是三位前PayPal（全球最大的在线支付平台）的员工：
 * 查德·赫利（Chad Hurley）
 * 陈士骏（Steve Chen）
 * 贾德·卡林姆（Jawed Karim）

在YouTube成立后迅速获得巨大成功，谷歌公司看到了其巨大的潜力，于 2006年10月 以 16.5亿美元
的股票交易将其收购。这在当时是一笔轰动性的收购，现在看来则是科技史上最成功的投资之一。

## 为什么叫YouTube
这个名字起得非常巧妙，完美地概括了平台的核心理念。我们可以把它拆成两部分来看：

   1. You（你）
      这部分很简单，直接指向了用户本身。它强调了这个平台的核心是用户生成内容（User-Generated
  Content）。意思是，这个平台是为你准备的，让你来创造、分享和广播你自己的视频。它体现了一种“人人都是创作者”的民主化精神。

   2. Tube（管子）
      这部分是理解这个名字的关键，带有一点历史和俚语的色彩。"Tube" 是对“电视（Television）”的旧式俚语称呼。
       * 来源： 这个说法的来源是老式的阴极射线管（Cathode-Ray Tube,
         CRT）电视机。在平板电视出现之前，电视机都非常笨重，其核心显像部件就是一个巨大的真空电子管。因此，人们就用“管子（the tube）”来代指电视机。在英语里，老一辈的人可能会说
         "What's on the tube tonight?" 来询问“今晚电视上有什么节目？”

  结合起来：

  YouTube = You + Tube = 你的电视机

  所以，这个名字的含义就是“你的个人电视台”或“你来播放的电视”。

  它精准地传达了这样一个革命性的想法：任何普通人都可以拥有自己的“频道”，向全世界播放自己的内容，而不再需要依赖传统的、由少数大公司控制的广播电视网络。这个名字在2005年那个时
  代，极具前瞻性和吸引力。
  
# Bilibili

## 和YouTube 同类别的网站有哪些，全球竞品？

当然。虽然从用户量、内容库和全球影响力来看，没有一个网站能与YouTube完全匹敌，但它的竞争对手可以根据不同的领域和市场进行划分。

  以下是YouTube在全球范围内的主要竞品类别和具体平台：

  1. 直接的综合视频平台 (Direct Competitors)

  这些平台在功能和模式上与YouTube最为相似，都提供长、短视频上传、分享和观看服务，但规模通常小得多。

   * Vimeo:
       * 特点: 成立于2004年，比YouTube还早。Vimeo定位更偏向于专业创作者和高质量内容，如独立电影、动画、商业广告等。它提供更优质的播放器和无广告的观看选项，是很多影视专业人士
         和企业的首选。
       * 竞争领域: 高质量视频托管和分享。

   * Dailymotion:
       * 特点: 来自法国，是欧洲最流行的视频网站之一。其界面和功能与YouTube非常相似，内容也包罗万象，但用户基础和内容库远小于YouTube。
       * 竞争领域: 综合视频分享（尤其在欧洲市场）。

   * Rumble:
       * 特点: 近年来兴起的一个平台，以其对言论自由的标榜和对创作者更友好的收入分成模式吸引用户。它在北美，尤其是在政治保守派用户中非常受欢迎。
       * 竞争领域: 综合视频分享，尤其在特定政治光谱的用户中。

  2. 短视频社交平台 (Short-Form Video)

  这是目前对YouTube用户时长和广告收入构成最大威胁的领域，YouTube为此推出了“Shorts”功能来直接对抗。

   * TikTok (抖音国际版):
       * 特点: 全球短视频领域的绝对王者。其强大的推荐算法和易于使用的创作工具，使其在年轻用户中拥有无与伦比的吸引力。
       * 竞争领域: 用户注意力、短视频内容、年轻用户群体。

   * Instagram Reels:
       * 特点: Meta (Facebook) 旗下的短视频功能，深度整合在Instagram内部。它利用Instagram庞大的现有用户基础，迅速成为TikTok的主要竞争对手。
       * 竞争领域: 短视频内容、广告收入。

  3. 直播平台 (Live Streaming)

  YouTube在游戏直播和事件直播领域投入巨大，但面临着非常专注和强大的竞争对手。

   * Twitch:
       * 特点: 亚马逊旗下，是全球游戏直播领域的领导者。它拥有非常成熟的社区文化、订阅和打赏系统，是绝大多数游戏主播的首选平台。
       * 竞争领域: 游戏直播、主播资源。

   * Kick:
       * 特点: 一个新兴的直播平台，以对主播极其有利的收入分成（95%给主播）而闻名，已经成功从Twitch和YouTube挖走了一些顶级主播。
       * 竞争领域: 游戏直播、顶级主播资源。

  4. 区域性的巨头 (Regional Giants)

  在一些国家，由于政策或文化原因，本土视频网站占据主导地位。

   * Bilibili (哔哩哔哩):
       * 国家: 中国
       * 特点:
         被称为“中国的YouTube”。从ACG（动画、漫画、游戏）文化起家，现已发展成为一个包含生活、科技、知识、时尚等内容的综合性视频社区，拥有独特的弹幕文化和强大的社区粘性。
       * 竞争领域: 在中国市场完全取代了YouTube的功能。

   * Youku (优酷) / iQIYI (爱奇艺) / Tencent Video (腾讯视频):
       * 国家: 中国
       * 特点: 这些平台更像是“YouTube + Netflix”的结合体。它们既有用户上传的内容（UGC），也有大量购买版权的电影、电视剧和自制综艺节目。
       * 竞争领域: 在中国市场的视频观看时长。

   * Rutube:
       * 国家: 俄罗斯
       * 特点: 俄罗斯本土的视频平台，在YouTube在俄受到限制后，得到了更多的官方支持和用户增长。
       * 竞争领域: 在俄罗斯市场的视频服务。

  总结表格

```sh
  ┌─────────────────┬──────────────────────────────┬─────────────────────────┐
  │ 平台 (Platform) │ 主要特点                     │ 与YouTube的主要竞争领域 │
  ├─────────────────┼──────────────────────────────┼─────────────────────────┤
  │ Vimeo           │ 高质量、专业内容、无广告选项 │ 专业视频托管            │
  │ Dailymotion     │ 欧洲流行、功能类似           │ 综合视频（欧洲）        │
  │ TikTok          │ 短视频王者、强大算法         │ 短视频、用户时长        │
  │ Instagram Reels │ 整合于IG、用户基数大         │ 短视频、广告收入        │
  │ Twitch          │ 游戏直播第一平台             │ 游戏直播                │
  │ Bilibili        │ 中国的YouTube、弹幕文化      │ 中国市场                │
  └─────────────────┴──────────────────────────────┴─────────────────────────┘
```

## Bilibili是抄袭的YouTube吗？Bilibili是什么时间成立的，分析一下Bilibili的产品以及背后的公司

1. Bilibili是抄袭的YouTube吗？

  简单回答：不是。更准确地说，Bilibili最初的灵感来源和模仿对象是日本的Niconico，而不是YouTube。

  这是一个非常关键的区别，也是理解B站核心文化的钥匙。

   * 灵感来源 (Niconico): Bilibili在初期几乎是Niconico的一个翻版。Niconico是日本一个著名的视频网站，其最核心、最标志性的功能就是 “弹幕（Danmaku）”——即评论会像子弹一样从屏幕右
     侧飞过，并与视频内容在时间上同步。这种形式创造了一种“大家在一起看”的强烈社区感和实时互动感。Bilibili完美地复刻并优化了这一体验。

   * 与YouTube的区别:
       * 核心不同: YouTube的核心是内容分发和算法推荐，它是一个巨大的、高效的全球视频搜索引擎和分发机器。而Bilibili的核心是社区和文化，它首先是一个围绕特定兴趣（最初是ACG）聚
         集起来的社区，视频是社区成员交流的载体。
       * 演变路径: 随着发展，Bilibili为了“破圈”（扩大用户群体），确实在功能和内容上借鉴了YouTube的很多成功模式，比如引入生活区Vlog、知识区内容、科技测评等，使其看起来越来越像
         一个“中国的YouTube”。
       * 结论: Bilibili的“基因”来自Niconico，它的“骨架”是社区。后来为了成长，它穿上了和YouTube类似的“外衣”。所以，它是一个独特的融合体，而非简单的抄袭。

  2. Bilibili是什么时间成立的？

   * Bilibili的前身是一个名为 Mikufans.cn 的网站。
   * 由创始人徐逸（网名：⑨bishi）于 2009年6月26日 创建。
   * 在 2010年1月24日，正式更名为Bilibili。

  3. Bilibili的产品分析

  Bilibili的产品形态非常丰富，可以看作是一个“视频+社区+游戏+直播+电商”的综合体。

  核心支柱：

   1. PUGV (Professional User Generated Video - 专业用户生成视频):
       * 这是B站的内容基石和护城河。与抖音的UGC（用户生成内容）不同，B站的UP主（视频创作者）创作的内容通常更长、制作更精良、信息密度更高。
       * 涵盖范围极广：从最初的游戏、动画杂谈，扩展到生活Vlog、美食探店、数码评测、科普知识、时尚美妆、鬼畜搞怪等所有领域。

   2. 弹幕文化与社区氛围:
       * 弹幕系统: 如前所述，这是B站的灵魂。它不仅是评论，更是梗的来源、情感的共鸣和身份的认同。
       * “正式会员”考试: 这是B站一个天才的设计。新用户需要通过一个包含100道题（涉及社区礼仪、弹幕文化、ACG知识等）的考试才能成为“正式会员”，拥有发弹幕、评论等全部权限。这个门
         槛极大地筛选了用户，保证了社区氛围的纯净和高质量。
       * 独特的社区文化: “UP主”、“一键三连”（点赞、投币、收藏）、“下次一定”、“镇站之宝”等都是深入人心的社区黑话和文化符号。

   3. OGV (Occupationally Generated Video - 专业机构生成视频):
       * 这是B站用来“破圈”和吸引新用户的重要手段。
       * 番剧（日本动画）: B站是中国最大的正版日本动画播放平台，这是其ACG核心用户的基本盘。
       * 纪录片与综艺: B站出品了大量高质量的纪录片（如《人生一串》）和综艺（如《说唱新世代》），在全网获得了极高的口碑。
       * 国创（国产动画）: 大力扶持国产动画，出品了如《灵笼》、《凡人修仙传》等热门作品。

   4. 直播 (Live Streaming):
       * 一个重要的内容补充和营收来源，尤其在游戏直播和虚拟主播（VTuber）领域非常强势。

   5. 商业化 (Monetization):
       * 游戏业务: 在很长一段时间里，这是B站的第一大收入来源。通过代理和联运热门手机游戏（如《Fate/Grand Order》）获得巨额收入。
       * 增值服务: 即“大会员”，用户付费后可以观看高质量的OGV内容，享受更高码率的视频等。
       * 广告: B站的广告相对克制，形式也更多样化，力求不破坏社区氛围。
       * 电商: “会员购”平台，主要销售动漫周边、手办、模型等，与核心用户群高度契合。

  4. 背后的公司分析

   * 公司主体: 上海幻电信息科技有限公司 是Bilibili在中国内地的主要运营实体。
   * 创始人与核心人物:
       * 徐逸 (⑨bishi): 创始人，B站的缔造者，目前更多是精神领袖。
       * 陈睿: 现任董事长兼CEO，B站的实际掌舵人。他曾是金山软件高管、猎豹移动联合创始人，在2011年作为天使投资人投资了B站，并于2014年正式以合伙人身份加入。陈睿的加入为B站带来
         了成熟的管理经验和商业化视野，是B站从一个个人网站走向上市公司的关键人物。
   * 上市情况:
       * Bilibili是一家上市公司。
       * 2018年3月28日，在美国纳斯达克 (NASDAQ) 挂牌上市，股票代码：BILI。
       * 2021年3月29日，在香港交易所 (HKEX) 完成二次上市，股票代码：9626。

  总结来说，Bilibili是一个从亚文化圈子成长起来的、具有极强社区粘性的内容平台。它的成功在于精准地抓住了中国年轻一代用户的文化需求，并通过独特的社区运营手段和不断“破圈”的内容
  策略，建立了一个难以被复制的、充满活力的生态系统。

## 如果说在全球找一家和YouTube最像的公司，你会选择谁，理由是什么？

如果必须在全球范围内选择一家和YouTube最像的公司，我会选择 Bilibili (哔哩哔哩)。

  虽然没有任何一家公司是YouTube的完美1:1复刻，但Bilibili是唯一一个在功能、内容生态、商业模式和文化影响力等多个维度上，与其在自身核心市场（中国）的地位和作用高度相似的平台。

  以下是详细的理由：

  理由一：视频内容的“万能杂货铺”

  两者都是内容极其丰富的综合性视频平台，几乎涵盖了所有你能想到的领域。

   * YouTube: 你可以在上面找到从美妆教程、数码评测、旅行Vlog、烹饪教学、教育课程到音乐MV和政治评论的所有内容。它是一个满足用户所有视频需求的“一站式商店”。
   * Bilibili:
     同样地，B站也拥有“生活区”、“科技区”、“知识区”、“美食区”、“鬼畜区”等无数分区，UP主们创作的内容包罗万象。一个中国用户几乎所有长视频的观看需求，都可以在B站得到满足。

  理由二：繁荣且核心的创作者生态 (YouTubers vs. UP主)

  两家公司的命脉都掌握在它们平台上的内容创作者手中。

   * YouTube: “YouTuber”已经成为一种公认的职业。平台围绕创作者建立了一套成熟的生态系统，包括广告分成、频道会员、Super Chat打赏、商品货架等，让创作者可以获得可观的收入。
   * Bilibili:
     “UP主”（Uploader的缩写）同样是B站的核心。B站也为UP主提供了“充电计划”（打赏）、广告（花火平台）、直播收入、悬赏带货等多种变现方式。无数全职UP主依靠在B站创作内容为生。

  理由三：对一代人拥有定义性的文化影响力

  两者都不仅仅是工具，更是定义了一代年轻人网络生活方式和流行文化的策源地。

   * YouTube: 全球的流行梗（Memes）、网络挑战、热门话题和网络红人，绝大多数都源自或发酵于YouTube。它是西方Z世代获取信息、娱乐和形成身份认同的核心场域。
   * Bilibili: B站同样是中国Z世代的文化大本营。无数的网络热词、鬼畜视频、弹幕梗都诞生于此。B站的社区文化和价值观深刻地影响着中国年轻人的思考和交流方式。

  理由四：高度相似的商业化路径

  为了实现增长和盈利，两者的商业模式也越来越趋同。

   * 广告: 两者最主要的收入来源都是视频广告。
   * 增值服务: YouTube有YouTube Premium（免广告、后台播放、独家内容），B站有大会员（看高码率视频、看独家番剧和影视）。
   * 直播与打赏: YouTube有直播功能和Super Chat/Super Thanks，B站有功能更强大的直播分区和“舰长”等打赏体系。
   * 自制内容: YouTube曾推出YouTube Originals，B站则大力投资纪录片、国创动画和综艺，逻辑如出一辙。

  ---

  为什么不是其他公司？

   * Vimeo/Dailymotion: 规模和文化影响力完全无法与YouTube相提并论。
   * TikTok: 核心是短视频和算法推荐，与YouTube以长视频和搜索为基础的媒体库模式有根本不同。
   * Netflix: 是专业影视内容（OGV）平台，而YouTube的核心是用户生成内容（UGC/PUGV）。

  结论：

  尽管Bilibili的弹幕文化、社区准入门槛（考试）和源自ACG的亚文化基因使其与YouTube在“气质”上有所不同，但从它在市场中所扮演的角色、所承载的功能以及所构建的生态系统来看，Bilibi
  li无疑是这个星球上与YouTube最相似的“镜像存在”。
